# 🐅 TechnoLAND: Индия 2026

> **В мистической Индии 2026 года** существует легенда о трёх забытых мастерских Панчамахабхута — конструирования, программирования и менеджмента, где технологии когда‑то гармонично сочетались с пятью природными стихиями (Земля, Вода, Огонь, Воздух и Эфир).

## 📖 О проекте

**TechnoLAND** — это интерактивный симуляционный хаб, погружающий пользователя в атмосферу технологических вызовов будущего на примере развивающейся Индии. Игроку предстоит решить проблемы, порожденные нарушением гармонии природных стихий, развивая технологии в трех направлениях:
* 🔧 **Конструирование** (Маскот: Конструктиус)
* 💻 **Программирование** (Маскот: Кодиус)
* 📈 **Экономика** (Маскот: Экономикус)

Приложение включает в себя тестирование предрасположенностей пользователя, глубокий лор и интерактивные симуляционные столы (моделирование, кодинг, управление экономикой).

## 🚀 Стек технологий

Проект построен на современном стеке фронтенд-технологий, обеспечивающем высокую скорость разработки и превосходный пользовательский опыт:
* **Фреймворк:** [React 18](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
* **Сборщик:** [Vite](https://vitejs.dev/)
* **Стилизация:** [Tailwind CSS](https://tailwindcss.com/) + Vanilla CSS (Glassmorphism и анимации)
* **Иконки:** [Lucide React](https://lucide.dev/)

## 🛠️ Установка и запуск

Для запуска проекта локально вам понадобятся [Node.js](https://nodejs.org/) и `npm`.

1. **Перейдите в директорию проекта:**
   ```bash
   cd project
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```

3. **Запустите сервер разработки:**
   ```bash
   npm run dev
   ```
   *Приложение будет доступно по адресу (обычно `http://localhost:5173`).*

4. **Продакшен сборка:**
   ```bash
   npm run build
   ```

## 🎨 Особенности дизайна

- **Glassmorphism & Cyber-vibe**: Использование стилей "матового стекла", свечения элементов интерфейса и теплых цветовых акцентов (оранжевый/amber).
- **Cinematic Transitions**: Плавные, кинематографичные анимации при переходах между экранами, эффекты печатной машинки в диалогах (Typewriter).
- **Многоуровневый интерфейс**: Профессиональный интерфейс с карточками, интерактивными элементами и продуманной типографикой.
